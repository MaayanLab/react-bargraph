# react-bargraph

A React implementation of a bar chart intended for use in Signature Commons Graph.

#### Usage
See [src/demo.jsx](./src/demo.jsx) for more info.

#### Development
```bash
npm start
```

#### Deployment
```bash
npm run build:lib
```
